=== IPv4 vs IPv6
[cols="^2,<6,<6", %autowidth.stretch]
|===
^|  ^| IPv4 ^| IPv6

^| 주소 크기
a| * 32bit
a| * 128bit

^| 헤더 크기
a| * 가변 길이
a| * 고정 길이 (40byte) + 확장 헤더

^| 헤더 길이
a| * 4bit
a| * 불필요

^| 표기법
a| * 8bit(10진수) &times; 4 +
123.456.789.000
a| * 16bit(16진수) &times; 8 +
1234:5678:...:ABCD:00FF

^| 전송 방식
a| * 멀티캐스트, 유니캐스트, 브로드캐스트
a| * 멀티캐스트, 유니캐스트, 브로드캐스트

^| 품질 제어
a| * 품질 보장 X
a| * 확장 헤더에서 지원
* 등급별 / 서비스별 패킷 구분 품질 보장

^| 보안 기능
a| * IPSec 프로토콜 추가
a| * 확장 헤더에서 기본 제공

^| 기타 기능
a| * X
a| * 모바일 IP, 웹캐스팅

^| 주소 할당
a| * A, B, C, D 클래스 비순차 할당 + subnet
a| * 네트워크 규모 및 단말기 수에 따라 순차 할당

^| 주소 갯수
a| * 약 43억개
a| * 약 31조개
|===
