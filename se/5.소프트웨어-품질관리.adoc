*#기신사# #효유이# #호보#*

__기__능 적합성 Functionality Suitability::
기능 성숙도 Functional Completeness::: 사용자의 목적 만족
기능 정확도 Functional Correctness::: 정확한 결과 제공
기능 타당성 Functional Appropriateness::: 목적의 완수를 용이

__신__뢰성 Reliability::
성숙도 Maturity::: 고장을 회피하며 신뢰도 충족
오류 허용성 Fault Tolerance::: 결함이 존재해도 성능 수준을 유지
회복 가능성 Recoverability::: 고장 발생 시 성능 및 데이터 회복
가용성 Availability:::

__사__용성 Usability::
이해 용이성 Understandability::: 요구에 적절한지 식별 가능
학습성 Learnability::: 사용자가 사용방법 학습 가능
운용성 Operability::: 제어 및 동작
사용자 인터페이스 미학 User Interface Aesthetics::: 이뿐가?
사용자 오류 보호 User error Protection::: 오류로 부터 사용자 보호
접근성 Accessibility::: 사용자 특성(나이/장애)에 상관 없이 사용

실행 __효__율성 Performance Efficiency::


__유__지보수성 Maintainability::
* 품질 저하 없이 수정 가능
* 변경이 다른 부분에 미치는 영향 최소화
* 변경에 대한 영향 분석 용이
* 테스트 용이
__이__식성 Portability:: 다른 제품을 대체 할 수 있어야 한다.

__호__환성 Compatibility:: 다른 제품과 자원 공유. 다른 제품에 악역향이 없어야 한다.
